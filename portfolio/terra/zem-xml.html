Обмінний файл XML

Обмінний файл XML земельної ділянки - файл, який містить всю інформацію о земельній ділянці (адреса, власник, цільове призначення ділянки, площа, координати та інші) для внесеня до Державного земельного кадастру.

Обмінний файл створюється сертифікованим інженером-землевпорядником та обов’язково завіряється електронним цифровим підписом останнього.


Документи необхідні для виготовлення обмінного файлу XML:

  • копія паспорту;
  • копія індивідуального податкового номеру;
  • установчі документи для юридичних осіб;
  • документи, що підтверджують право власності на землю;
  • документи, що засвідчують право користування земельною ділянкою.

Зміст обмінного файлу та його технічні характеристики повинні відповідати вимогам Порядку ведення Державного земельного кадастру, затвердженого Постановою № 1051 від 17.10.2012 р.


Обмінний файл XML створюється:

  • первинна реєстрація земельних ділянок (відведення, поділ, кприсвоєння кадастрового номеру);
  • виправлення помилок в даних про земельну ділянку;
  • внесення сервітутів або обмежень на земельні ділянки.
